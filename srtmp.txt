Структура файла для шаблонной замены. Входной файл построчно обрабатывается каждым шаблоном. Если соответствие найдено, 
в выходной поток выводится замена, после чего обрабатывается следующая строка исходного файла. Таким образом одна строка 
может быть обработана только одним шаблоном. Соответствующая регулярному выражению (шаблону) строка исходного файла может 
заменяться на одну или несколько строк.

Несколько строк заголовка (до первого шаблона),
один раз копируемого в выходной поток

/шаблон/замена
замена
замена
/
/regexp/regreplace
regreplace
regreplace
/

regreplace/

Несколько завершающих строк (после последнего шаблона) один, 
раз копируемых в выходной поток.
